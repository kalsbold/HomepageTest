네트워킹 게임연구실이란?

IoT, Cloud Computing, Wireless Sensor Network 환경에서의 연구를 진행하고 있습니다.

  • IoT(Internet of Things)

    사물 인터넷(Internet of Thing)은 각종 사물에 센서와 통신 기능을 내장하여 인터넷에 연결하는 기술을 의미한다.

  • Cloud Computing

    클라우드 컴퓨팅(cloud computing)은 인터넷 기반(cloud)의 컴퓨팅(computing) 기술을 의미한다.

  • Wireless Sensor Network

    무선 센서 네트워크(wireless sensor network)는 센서를 네트워크로 구성한 것을 말한다.
